Learn How to Use PPP

Authentication

PPP (point-to-point protocol) authentication is possible through one of two methods. PAP (Password Authentication Protocol) and CHAP (Challenge Handshake Authentication Protocol). Of the two, PAP is the easiest to implement and understand but has it's limits due to the fact that it transmits it's password in clear text. CHAP on the other hand uses a more secure algorithm that does not send passwords. You can read more about the methods in RFC 1334.Â  The video below uses two routers to demonstrate how this is done.
